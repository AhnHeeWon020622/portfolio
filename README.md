# 안희원 포트폴리오

## 프로젝트 1

🔗[**공공의료 데이터 기반 병원·응급실·예약 통합 관리 시스템**](https://github.com/AhnHeeWon020622/CareBridge)

## 개요

- 기존 의료 서비스의 정보 제한 문제를 개선하기 위해 **응급실 정보·병원 예약을 하나로 통합한 플랫폼을 설계·구현**하였습니다.
- 공공의료 API 데이터를 정제·가공하고**필터링·점수화 로직을 적용하여 사용자 상황에 맞는 응급실을 신속하게 선택할 수 있도록 구현**하였습니다.


## 담당 역할 및 기술 스택

- **역할** : 실시간 응급실 조회 페이지 구현
- **프론트** : HTML, CSS, JavaScript(AJAX), Jquer
- **백엔드** : Django
- **DB** : MySQL, Django ORM
- **서버** : AWS EC2
- **API** : Kakao Map API, 공공데이터포털 API(XML)
- **형상관리** : GitHub, gitFlow방식 관리
- **개발툴** : VS Code
